## qtile key bindings

```text
super + Tab                           Toggle through layouts
super + w                             Keyboard focus screen(0)
super + e                             Keyboard focus screen(1)
super + r                             Keyboard focus screen(2)
super + k                             Switch between windows in current stack pane
super + j                             Switch between windows in current stack pane
super + n                             Restore all windows to default size ratios
super + m                             Toggle a window between minimum and maximum sizes

super + Ctrl + k                      Move up a section in treetab
super + Ctrl + Enter                  Toggle between split and unsplit sides of stack

super + Shift + k                     Move windows down in current stack
super + Shift + j                     Move down a section in treetab
super + Shift + l                     Grow size of current window
super + Shift + h                     Shrink size of current window
super + Shift + left                  Move window to workspace to the left
super + Shift + right                 Move window to workspace to the right
super + Shift + Enter                 Toggle floating
super + Shift + Space                 Swap panes of split stack

super + Shift + Esc                   Shutdown Qtile
super + Shift + c                     Kill active window
super + Shift + r                     Restart Qtile

```
